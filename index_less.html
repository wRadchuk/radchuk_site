Использую заголовок второго уровня для разделения информации

Коротко о CSS - это каскадная таблица стилей для украшеня веб-страницы. В CSS теги называются селекторами и если нужно украсить тег, то используем одноименный селектор.

В CSS есть объектная часть ООП, есть селекторы классов и селекторы индексов, а так же селекторы групп.

Всё это не сложно запомнить, так как имена классов и индексов задаём сами. Единственное важное h1 {стиль} - обычный селектор заголовка первого уровня, .your_class {стиль} - селектор класса, #your_id {стиль} - селектор индекса и h1, .your_class, #your_id {стиль} - пример селектора групп в который я добавил сразу три типа селекторов и можно для всех них указать один стиль. Например эти селекторы отвечают за выводимый текст в разных местах веб-страници, значит мы можем указать им шрифт, цвет, размер шрифта и прочие свойства.

Если мы укажем для тега h1 цвет текста на красный, а потом ниже укажем для него же цвет на синий, то выведедется текст синим цетом - последовательность вжна. Если зададим размер шрифта для селектора html, а потом укажем размер шрифта для селектора body, то шрифт дочерних елементов будет увеличен сначало родительским селектором html, а затем ещё раз дочерним его селектором bod- s = (s * 100%) + (s * 120%) + (s * 140%)
